325 - SoCal Installation

SCE - ESA Whole Home (PP/D)

What this form is for

Installation Form 325 for SoCal Whole Home Program. It is organized into 31 sections, covering areas such as New Central HVAC System, Water Heater Information, Insulation Area, and Clothes Dryer & Washer. It connects to two other forms and looks up reference data from 29 related forms.

What happens automatically

Three automated steps watch this form — they run when a saved record changes.

Ready for Review325_READY_FOR_REVIEW
Runs when a 325 - SoCal Installation record is updated, and only if Installation Status is 'Pending Review'.
Sends a notification.
Sends an email to operations@maromaes.com — subject “SoCal Whole Home Installation Review for {the record's ESA Key}”.
Installation Rejected325_INSTALLATION
Runs when a 325 - SoCal Installation record is updated, and only if Review Status is 'Corrections Required'.
Sends an email to ProjectContactEmail; CC: operations@maromaes.com — subject “SoCal Whole Home Installation Requires Corrections - {the record's Enrollment Number}”.
Review Notification (Operations)325_REVIEW_NOTIFICATION
Runs when a 325 - SoCal Installation record is updated, and only if Installation Status compares to 'None' and Reviewer Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | 325 - {the record's Enrollment Number}”.
