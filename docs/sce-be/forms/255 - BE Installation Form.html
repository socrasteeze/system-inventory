255 - BE Installation Form

SCE - Building Electrification

What this form is for

Installation Form 255 for the Building Electrification Program. It is organized into 29 sections, covering areas such as New Central HVAC System, Minor Home Repair, Water Heater Information, and Electrical. It connects to two other forms and looks up reference data from two related forms.

What happens automatically

Four automated steps watch this form — they run when a saved record changes.

Installation Desktop Review255_INSTALLATION_DESKTOP
Runs when a 255 - BE Installation Form record is updated, and only if Submission Status is 'Ready for Review'.
Sends an email to vbecerra@maromaes.com — subject “Building Electrification Installation Review for {the record's Full Name} {the record's Enrollment Number}”.
Installation Approved255_INSTALLATION
Runs when a 255 - BE Installation Form record is updated, and only if Review Status is 'Approved'.
Sends an email to ReviewContact; CC: operations@maromaes.com — subject “Building Electrification Installation Approved for {the record's Full Name}”.
Installation Requires Corrections255_INSTALLATION_2
Runs when a 255 - BE Installation Form record is updated, and only if Review Status is 'Corrections Required'.
Sends an email to ContactEmail; CC: @LastModifier.Email — subject “BE Installation Requires Corrections for {the record's Full Name}”.
Review Notification (Operations)255_REVIEW_NOTIFICATION
Runs when a 255 - BE Installation Form record is updated, and only if Installation Status compares to 'Draft' and Reviewer Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | 255 - {the record's Enrollment Number}”.
