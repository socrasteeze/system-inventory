What this form is for

This form is one of the working forms staff complete as part of SDGE - ESA Whole Home (PP/D). It connects to two other forms and looks up reference data from three related forms.

What happens automatically

Four automated steps watch this form — they run when a saved record changes.

Installation Ready for ReviewINSTALLATION
Runs when a 425 - San Diego Installation record is updated, and only if Submission Status is 'Ready for Review'.
Sends an email to operations@maromaes.com — subject “SDGE Whole Home Installation Review for {the record's Customer Name} - {the record's Enrollment Number}”.
Installation ApprovedINSTALLATION_2
Runs when a 425 - San Diego Installation record is updated, and only if Review Status is 'Approved'.
Sends an email to {CreatorEmail},{ProjectContactEmail} — subject “SDGE Whole Home Installation Approved {the record's Enrollment Number}”.
Installation RejectedINSTALLATION_3
Runs when a 425 - San Diego Installation record is updated, and only if Review Status is 'Corrections Required'.
Sends an email to {CreatorEmail},{ProjectContactEmail} — subject “SDGE Whole Home Installation Requires Corrections - {the record's Enrollment Number}”.
Review Notification (Operations)REVIEW_NOTIF_3
Runs when a 425 - San Diego Installation record is updated, and only if Installation Status compares to 'None' and Reviewer Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | 425 - {the record's Enrollment Number}”.
