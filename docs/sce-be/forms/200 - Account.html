SCE - Building Electrification

What this form is for

View and manage all currently customers within the Building Electrification Program for Southern California. This form is the central record in SCE - Building Electrification — most other forms connect back to it. It is organized into 12 sections, covering areas such as Permits, Site Information, Project Information, and Demographics. It connects to four other forms and looks up reference data from three related forms.

What happens automatically

Two automated steps watch this form — they run when a new record is created.

Customer Interest Receipt200_CUSTOMER_INTEREST
Runs when a new 200 - Account record is created, and only if Send Email Confirmation Receipt is 'True'.
Sends an email to AccountEmail — subject “Building Electrification Program”.
Create Enrollment Form200_CREATE_ENROLLMENT
Runs when a new 200 - Account record is created, and only if Lead Status is 'Completed Lead' and Project Stage is 'Enrollment' and Project Status is 'In Progress'.
Creates a new 210 - Enrollment & Assessment Form record, filling in three fields automatically (skips if one already exists).
