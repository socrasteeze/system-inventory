Assessment Intake - Form 02

Low-Income Weatherization Program

What this form is for

LIWP FWHC Assessment Intake to evaluate all applicable measures in dwelling. Assessment Intake - Form 02 is a working form in the Low-Income Weatherization Program process. It connects to two other forms and looks up reference data from two related forms.

What happens automatically

Five automated steps watch this form — they run when a saved record changes.

Desktop ReviewDESKTOP_REVI_2
Runs when a Assessment Intake - Form 02 record is updated, and only if EE Submission Status compares to 'None'.
Sends an email to operations@maromaes.com — subject “LIWP FWHC | Assessment Review for {the record's Enrollment Number}”.
Approved - Issue Work OrderAPPROVED_-_I
Runs when a Assessment Intake - Form 02 record is updated, and only if EE Review Status is 'Enrollment Approved'.
Sends an email to operations@maromaes.com — subject “LIWP FWHC | Issue Work Order {the record's Enrollment Number}”.
Corrections RequiredCORRECTIONS_
Runs when a Assessment Intake - Form 02 record is updated, and only if EE Review Status is 'Corrections Required' or SPV Review Status is 'Corrections Required'.
Sends an email to {ContactEmail} — subject “LIWP FWHC | Assessment Requires Corrections {the record's Enrollment Number}”.
Review Notification (Operations) - EEREVIEW_NOTIF_2
Runs when a Assessment Intake - Form 02 record is updated, and only if Submission Status compares to 'None' and EE Review Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | Form 02- {the record's Enrollment Number}”.
Review Notification (Operations) - SPVREVIEW_NOTIF_3
Runs when a Assessment Intake - Form 02 record is updated, and only if Submission Status compares to 'None' and SPV Review Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | Form 02- {the record's Enrollment Number}”.
