What this form is for

Low-Income Weatherization Program's Inspection Form. It is organized into eight sections, covering areas such as Customer Information, Measure Details, Inspector Acknowledgement, and Customer Acknowledgement. It connects to two other forms and looks up reference data from two related forms.

What happens automatically

Four automated steps watch this form — most run when a saved record changes (two more are currently switched off).

New Work Order CreatedINSPECTION_NEW_WORK_ORDER
Runs when a new Inspections record is created.
Sends an email to mayracmany@gmail.com — subject “FW Inspection Work Order Created {the record's Inspection ID}”.
Note Added by InspectorINSPECTION_NOTE_ADDED_BY
Runs when a Inspections record is updated, and only if Note Reply compares to 'None'.
Sends an email to customerservice@maromaes.com — subject “Inspection Note for {the record's Enrollment Number}”.
Work Order Assigned to Inspector UserINSPECTION_WORK_ORDER_ASSIGNED
Runs when a new Inspections record is created, and only if Dispatch to Inspector is 'True' and Inspector Name compares to 'Open'.
Sends an email to {InspectorEmail} — subject “Inspection Work Order {the record's Inspection ID} Assigned”.
Inspection ResultINSPECTION_INSPECTION_RESULT
Runs when a Inspections record is updated, and only if Inspection Status compares to 'None'.
Sends an email to customerservice@maromaes.com — subject “FW Inspection {the record's Inspection ID} - {the record's Inspection Status}”.
Notify Contractor on Failed MeasuresINSPECTION_NOTIFY_CONTRACTOR_ON
Runs when a Inspections record is updated, and only if Post-Inspection Status is 'Fail'.
Sends an email to {InstallationContractorEmail} — subject “ACTION REQUIRED | LIWP Inspection {the record's Enrollment Number}”.
Currently switched off — it will not run.
Note AddedINSPECTION_NOTE_ADDED
Runs when a Inspections record is updated, and only if Note Reply compares to 'None'.
Sends a notification.
Currently switched off — it will not run.
