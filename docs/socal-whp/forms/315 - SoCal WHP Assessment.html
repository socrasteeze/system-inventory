315 - SoCal WHP Assessment

SCE - ESA Whole Home (PP/D)

What this form is for

Documents and evaluates diagnostic tests performed during a residential home assessment. It includes intake fields for recording results from the Blower Door Test, which measures air leakage and overall envelope tightness; the Duct Test & Sealing, which assesses duct system efficiency and verifies proper sealing; and Natural Gas Appliance Testing, which evaluates the performance, safety, and combustion characteristics of gas-powered appliances. Form 315 ensures accurate data collection to support compliance with program standards and safety protocols. It is organized into 34 sections, covering areas such as Pre-Installation NGAT, Pre-Installation Blower Door Test, Post-Installation Blower Door Test, and Site Information. It connects to one other form and looks up reference data from one related form.

What happens automatically

Five automated steps watch this form — they run when a saved record changes.

Desktop Review315_DESKTOP_REVIEW
Runs when a 315 - SoCal WHP Assessment record is updated, and only if Submission Status is 'Ready for Review'.
Sends an email to operations@maromaes.com — subject “SoCal Whole Home Assessment Review for {{ApplicantFullName}} - {the record's ESA Key}”.
Approved315_APPROVED
Runs when a 315 - SoCal WHP Assessment record is updated, and only if Review Status is 'Assessment Approved'.
Sends an email to AssessmentContractorEmail; CC: operations@maromaes.com — subject “SoCal Whole Home Assessment Approved for {{ApplicantFullName}} - {the record's ESA Key}”.
Submission Results315_SUBMISSION_RESULTS
Runs when a 315 - SoCal WHP Assessment record is updated, and only if Assessment Status compares to 'Corrections Required' and Reviewer Name compares to 'None'.
Sends an email to AssessmentContractorEmail; CC: operations@maromaes.com — subject “SoCal Whole Home 315 | Project {the record's Enrollment Number} {the record's Assessment Status}”.
Review Notification (Operations)315_REVIEW_NOTIFICATION
Runs when a 315 - SoCal WHP Assessment record is updated, and only if Assessment Status compares to 'None' and Reviewer Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | 315 - {the record's ESA Key}”.
Update Installation Contractor for Form 315300_UPDATE_INSTALLATION
Runs when a 300 - Account Management record is updated, and only if Project Stage is 'Installation' and Installation Status is 'In Progress'.
Updates the matching 315 - SoCal WHP Assessment record, filling in one field automatically.
