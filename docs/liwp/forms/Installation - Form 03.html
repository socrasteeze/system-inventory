Installation - Form 03

Low-Income Weatherization Program

What this form is for

This form is the central record in Low-Income Weatherization Program — most other forms connect back to it. It is organized into 32 sections, covering areas such as Electric Baseload Measures, Additional Measures, Duct Repair / Sealing / Replacement, and Infiltration Reduction - Shell Sealing. It connects to two other forms and looks up reference data from three related forms.

What happens automatically

Four automated steps watch this form — they run when a saved record changes.

Desktop ReviewINSTALLATI_DESKTOP_REVIEW
Runs when a Installation - Form 03 record is updated, and only if Submission Status is 'Ready for Review' and Review Status is at least 'Enrollment Approved'.
Sends an email to operations@maromaes.com — subject “LIWP FWHC | Installation Submitted {the record's Enrollment Number}”.
ApprovedINSTALLATI_APPROVED
Runs when a Installation - Form 03 record is updated, and only if Review Status is 'Enrollment Approved'.
Sends an email to {ContactEmail} — subject “LIWP FWHC | Installation Form Approved {the record's Enrollment Number}”.
Corrections RequiredINSTALLATI_CORRECTIONS_REQUIRED
Runs when a Installation - Form 03 record is updated, and only if Review Status is 'Corrections Required'.
Sends an email to {ContactEmail} — subject “LIWP FWHC | Installation Requires Corrections”.
Review Notification (Operations)INSTALLATI_REVIEW_NOTIFICATION
Runs when a Installation - Form 03 record is updated, and only if Installation Status compares to 'None' and Reviewer Name compares to 'None'.
Sends an email to notifications@maromaes.com — subject “[Auto-Log] Review Notification | Form 03 - {the record's Enrollment Number (Computed)}”.

Filling it out

There are 897 fields to work with; 287 must be filled in, and 105 only appear in certain situations. As you fill it out, 123 fields automatically update, reveal, or check other parts of the form.
